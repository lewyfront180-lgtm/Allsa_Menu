DESCRIPTION
Allsa Menu is a custom mod menu/loader designed for VR games such as Gorilla Tag VR and Animal Company. It provides a simple and clean interface that allows users to manage and launch supported modifications in an organized way.

The goal of Allsa Menu is to make mod handling more accessible and user-friendly, with a lightweight design focused on performance and ease of use. The menu is built to work smoothly inside VR environments while keeping everything straightforward and easy to navigate.

TUTORIAL
To get the Allsa Loader working correctly, you first need to run the Fixer tool on your PC. After launching it, you will be asked to locate and select the Allsa Loader file stored on your computer.

Make sure you choose the correct file from your device, as the Fixer uses it to properly prepare the loader. Once the file has been selected and processed, you can then start the Allsa Loader without any problems.

In short: run the Fixer → pick the Allsa Loader file from your PC → finish the process → then launch the Allsa Loader.
